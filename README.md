#Frontend Challenge


## Setup

### 1. Install dependencies

```sh
$ npm install
```

### 2. Install dependencies
```sh
$ npm start
```

### *  Run storybook
```sh
$ npm run storybook
```

### *  Buil
```sh
$ npm run build-dev
```

## Front-end Architecture

To develop the application i used atomic design methodology


### Atoms
    - Block
    - Button
    - Checkbox
    - ContentEditable
    - Heading
    - HorizontalRule
    - Icon
    - List
    - Paragraph
    - Spinner

### Molecules
    - IconButton
    - Modal

### Organisms
    - Header
    - TranscriptionList

### Templates
    - PageTemplate
### Page
    -HomePage
