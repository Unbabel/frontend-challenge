# frontend-challenge

## Installs all needed dependencies
```
npm install
```

### Runs the project
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Runs Unit Tests
```
npm run test:unit
```

### Runs e2e Tests
```
npm run test:e2e:headless
```

### Link to sample page
```
/sample-page
```
