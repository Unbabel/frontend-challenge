# Transcriptions App

## Description
This project has been made using Vue 3 and TypeScript 

## Set up
1. Make sure Node and npm are installed from the terminal. Steps:

    ```
    npm install
    ```

2. Start the project.

    ```
    npm run serve
    ```

3. Now you can visit http://localhost:8080 to view the client

4. To run the Tests: 

    ```
    npm run test:unit
    ```
